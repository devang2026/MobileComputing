# EXP 6 - WRITE TO DATABASE APP

## Aim
To develop an Android application that writes data to a database.

## Description
This application demonstrates storing data in a database (SQLite or Firebase) from an Android app.
It shows how to handle user input, connect to a database, and perform insert operations.

## Features
- Input data through form fields
- Save data to database
- View stored records
- Update/Delete records (optional)

## Output
Data entered by the user is successfully stored in the database and can be retrieved later.

## Result
Thus, the Write to Database App was developed and tested successfully.
