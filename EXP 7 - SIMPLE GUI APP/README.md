# EXP 7 - SIMPLE GUI APP

## Aim
To develop a simple Android application with a graphical user interface (GUI).

## Description
This app demonstrates basic GUI elements in Android such as buttons, text fields, and labels.
It shows how to handle user interactions and events.

## Features
- Buttons and text input
- Display output on screen
- Simple event handling

## Output
Users can interact with the GUI and see responses to their actions.

## Result
Thus, the Simple GUI App was developed and tested successfully.
