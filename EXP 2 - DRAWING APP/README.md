# EXP 2 - DRAWING APP

## Aim
To develop an Android application that allows users to draw on the screen using touch input.

## Description
This drawing application enables users to create freehand drawings on the screen.
It captures touch events and renders paths on a canvas.

## Features
- Touch-based drawing
- Clear / reset canvas
- Simple and interactive UI

## Output
Users can draw shapes and freehand sketches using touch gestures.

## Result
Thus, the Drawing App was developed and tested successfully.
