# EXP 9 - DRAW GRAPHICAL PRIMITIVES APP

## Aim
To develop an Android application to draw basic graphical primitives.

## Description
This application allows users to draw shapes like lines, rectangles, circles, and polygons.
It demonstrates handling of canvas, paint, and touch events in Android.

## Features
- Draw lines, rectangles, circles, and polygons
- Clear/reset canvas
- Change colors and stroke width (optional)

## Output
The application successfully renders the selected graphical primitives on the screen.

## Result
Thus, the Graphical Primitives App was developed and tested successfully.
