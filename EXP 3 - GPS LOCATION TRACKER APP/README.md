# EXP 3 - GPS LOCATION TRACKER APP

## Aim
To develop an Android application to retrieve and display the user's current GPS location.

## Description
This application uses GPS and network providers to fetch the user's latitude and longitude.
It demonstrates location permissions, location services, and real-time updates.

## Features
- Fetch current latitude and longitude
- Uses GPS provider
- Displays location on screen

## Output
The application successfully displays the user's current location coordinates.

## Result
Thus, the GPS Location Tracker application was developed and tested successfully.
