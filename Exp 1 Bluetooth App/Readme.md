# EXP 1 - BLUETOOTH FILE TRANSFER APP

## Aim
To develop an Android application for transferring files using Bluetooth.

## Description
This application enables file transfer between two Android devices using Bluetooth
connectivity. It demonstrates device discovery, pairing, and data transmission.

## Modules / Features
- Bluetooth enable / disable
- Device discovery
- File selection
- File transfer

## Output
Successful transfer of files between two Bluetooth-enabled devices.

## Result
Thus, the Bluetooth File Transfer application was developed and tested successfully.
