# EXP 10 - NOTIFICATION SIMULATION APP

## Aim
To develop an Android application that simulates notifications.

## Description
This application demonstrates how to create and display notifications on an Android device.
It includes features like simple notifications, custom messages, and notification actions.

## Features
- Generate simple notifications
- Custom notification messages
- Optional: notification with action buttons

## Output
The app successfully displays notifications as intended, simulating real-world scenarios.

## Result
Thus, the Notification Simulation App was developed and tested successfully.
