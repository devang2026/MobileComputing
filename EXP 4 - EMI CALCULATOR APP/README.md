# EXP 4 - EMI CALCULATOR APP

## Aim
To develop an Android application to calculate Equated Monthly Installment (EMI).

## Description
This application calculates EMI based on loan amount, interest rate, and loan tenure.
It demonstrates user input handling, mathematical computation, and result display.

## Features
- Input loan amount
- Input interest rate
- Input loan tenure
- EMI calculation and display

## Output
The application displays the calculated EMI based on the given inputs.

## Result
Thus, the EMI Calculator application was developed and tested successfully.
